## 1. 项目概述

### 这是什么？

**OpenClaw Agent 管理仪表盘** - 一个轻量级 Web UI，用于监控和管理多个 OpenClaw Bot/Agent 的运行状态。

### 核心功能

| 功能 | 描述 |
|------|------|
| **Bot总览** | 卡片墙展示所有Agent的模型、平台绑定、会话统计 |
| **模型管理** | 查看所有Provider和模型配置 |
| **会话管理** | 按Agent浏览会话，类型识别（私聊/群聊/定时） |
| **Token统计** | 消耗趋势图表（日/周/月） |
| **告警中心** | 配置告警规则，飞书通知 |
| **平台测试** | 一键测试飞书/Discord连通性 |
| **像素办公室** | Agent化身像素角色，在办公室里走动互动 🎮 |

### 技术栈

- **前端**: Next.js 16 + React 19 + TypeScript
- **样式**: Tailwind CSS 4
- **数据**: 直接读取 `~/.openclaw/openclaw.json`，无数据库
- **要求**: Node.js 18+
