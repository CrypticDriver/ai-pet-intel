## 5种推荐设计风格

### 1. 极简线条风格 (Minimalist Line Art)

**代表IP**: 线条小狗、LINE Friends
**设计特点**:
- 流畅简洁的2px线条
- 单色或双色配色（黑线+1-2个点缀色）
- 高度识别性的轮廓
- 简化到极致的面部表情（点状眼睛、小嘴）

**技术优势**:
- 文件体积小
- 容易做动画变体
- 易于批量生成表情包

**适合场景**: 聊天贴纸、轻量级App、治愈系应用

**Design参考要点**:
```
线条粗细: 2px
填充: 纯白或透明
主线条色: #3A3A3A 或 #333333
点缀色: 1个柔和色彩（如薄荷绿、粉色）
```

### 2. Kawaii/卡哇伊风格 (Japanese Cute)

**代表IP**: Hello Kitty、角落生物(Sumikko Gurashi)、懒懒熊(Rilakkuma)
**设计特点**:
- 超大的眼睛（占头部30%以上）
- 圆润柔和的形状，无锐角
- 极简面部：两个点眼睛 + 小嘴
- 低位置五官（模仿婴儿比例）
- 粉嫩腮红

**配色规范**:
- **Pop Kawaii**: 明亮饱和色
- **Pastel Kawaii**: 柔和粉彩色（粉、薄荷绿、天蓝、奶油黄）
- **Yume Kawaii**: 梦幻渐变（薰衣草紫、泡泡糖粉）

**技术优势**:
- 全球通用的"萌"感
- 触发天然的保护欲
- 强烈的情感连接

**Design参考要点**:
```
头身比: 2:1 到 3:1
眼睛: 大圆点，无瞳孔细节
颜色: 3-4色柔和色板
腮红: 高位圆形腮红是标配
```

---

### 3. Chibi/Q版风格 (Super-Deformed)

**代表IP**: Pokemon、手游角色、Nendoroid粘土人
**设计特点**:
- 极端夸张的头身比（1:1）
- 大头小身体
- 夸张的表情变化
- 保留角色关键特征（放大处理）

**配色规范**:
- 明亮饱和色为主
- 可以有渐变和阴影
- 比Kawaii更丰富的细节

**适合场景**: 游戏角色、收藏品、表情包

**Design参考要点**:
```
头身比: 1:1 或 1.5:1
头部: 大圆形，占整体50%以上
四肢: 短小粗壮，无关节细节
表情: 夸张，情绪外露
```

---

### 4. 扁平插画风格 (Flat Illustration)

**代表IP**: Duolingo Duo猫头鹰、Slack图标、Google Doodle
**设计特点**:
- 无渐变、无阴影的纯色填充
- 几何化的简洁形状
- 现代科技感
- 清晰的轮廓边缘

**配色规范**:
- 高饱和度的品牌色
- 有限色板（3-5色）
- 色块分明

**技术优势**:
- SVG友好，完美缩放
- 品牌识别度高
- 动画过渡简单

**Design参考要点**:
```
形状: 几何基础形状组合
颜色: 纯色填充，无渐变
轮廓: 可选（无轮廓或粗轮廓）
风格: 现代、干净、专业
```

---

### 5. 潮玩精灵风格 (Designer Toy/Art Toy)

**代表IP**: Pop Mart Labubu、Molly、Crybaby、Dimoo
**设计特点**:
- 独特的"丑萌"美学
- 有故事背景的角色设定
- 精致细节 + 可爱整体
- 略带诡异或反差的元素（如Labubu的尖牙）

**配色规范**:
- 更成熟的色彩
- 可以有渐变和质感
- 每个系列有独特配色主题

**技术优势**:
- 收藏价值感
- 差异化强
- 适合多系列延展

**Design参考要点**:
```
特点: 有"个性"而非单纯"可爱"
反差萌: 可爱外表+特殊元素（尖牙、大耳朵、独特表情）
质感: 可以有渐变、高光、材质感
系列化: 设计时考虑变体可能性
```
