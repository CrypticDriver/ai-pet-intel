## 1. 竞品社交功能分析

### 经典参考：Club Penguin

**Town Center（城镇中心）**:
- 开放式广场，玩家自由移动
- 是最繁忙的社交区域
- 连接多个功能区（咖啡店、夜总会、商店）
- 主要用于聚会和活动

**Plaza（广场）**:
- 连接Forest和Snow Forts
- 包含Pet Shop、Pizza Parlor等
- 玩家带着宠物（Puffles）互动

**技术特点**:
- Flash渲染（现已淘汰）
- 服务器分区（不同房间）
- 实时同步但有人数上限

---

### 经典参考：Habbo Hotel

**核心社交设计**:
- 酒店大堂作为中心广场
- 用户可创建自己的房间
- 访问他人房间是核心玩法
- 家具摆放+角色移动+聊天

**技术特点**:
- 等距视角（Isometric）
- 服务器房间管理
- 实时多人同步

---

### 现代参考：Tamagotchi Family / My Tamagotchi

**社交功能**:
- 访问好友的宠物
- 查看好友宠物信息
- 简单礼物交换
- **不是实时互动**，而是异步展示

---

### 现代参考：Talking Tom Friends

**社交功能**:
- 访问好友的游戏
- 查看好友养成进度
- 竞争排行榜
- **异步社交**为主

## 2. 技术方案分析

### 方案A：异步展示广场（MVP推荐）⭐

**概念**：
```
用户进入广场 → 看到其他用户的Pet快照 → 点击查看详情
```

**技术实现**：
```typescript
// 数据结构
interface PetSnapshot {
  petId: string;
  ownerId: string;
  ownerName: string;
  petName: string;
  petEmoji: string;
  mood: number;
  level: number;
  outfit: string[];     // 装扮
  lastOnline: Date;
  position: { x: number, y: number }; // 广场位置
}

// API
GET /api/plaza/pets  → 返回当前在广场的Pet列表
POST /api/plaza/join → 将自己的Pet放入广场
DELETE /api/plaza/leave → 离开广场
```

**工作量**：
| 任务 | 时间 | 负责 |
|------|------|------|
| 广场场景Canvas | 2-4h | Dev |
| Pet位置API | 2-4h | Dev |
| Pet点击查看 | 1-2h | Dev |
| 广场背景设计 | 2-4h | Design |
| **合计** | **8-14h** | - |

**优点**：
- ✅ 不需要WebSocket
- ✅ 服务器压力小
- ✅ 实现简单，风险低
- ✅ 能感受到"社交"氛围

**缺点**：
- ❌ 不是实时互动
- ❌ 不能看到Pet移动

---

### 方案B：轻量实时广场（v1.1建议）

**概念**：
```
用户进入广场 → 看到其他用户的Pet实时位置 → 简单互动（招手、送心）
```

**技术实现**：
```typescript
// WebSocket消息
interface PlazaMessage {
  type: 'join' | 'leave' | 'move' | 'emote';
  petId: string;
  data: any;
}

// 服务器广播
// 收到move → 广播给同广场所有用户
// 客户端每秒最多发送3次位置更新（节流）
```

**工作量**：
| 任务 | 时间 | 负责 |
|------|------|------|
| WebSocket服务 | 4-8h | Dev |
| 广场实时渲染 | 4-8h | Dev |
| Pet移动同步 | 4-8h | Dev |
| 表情/互动系统 | 2-4h | Dev |
| 广场背景设计 | 2-4h | Design |
| **合计** | **16-32h** | - |

**优点**：
- ✅ 能看到Pet实时移动
- ✅ 简单互动（招手、送心）
- ✅ 真正的"社交"感

**缺点**：
- ⚠️ 需要WebSocket基础设施
- ⚠️ 服务器资源消耗更高
- ⚠️ 复杂度显著增加

## 3. 广场场景设计建议

### 视觉风格

**参考Club Penguin的Plaza**：
- 露天场景
- 草地/石板路
- 树木、长椅、喷泉
- 明亮、温暖的氛围

### 建议场景元素

```
+-------------------------------------------+
|                   天空                      |
|        ☀️                                  |
|     🌳          喷泉 ⛲          🌳       |
|                                           |
|   长椅🪑    [Pet区域]    长椅🪑           |
|                                           |
|     🌸         草地          🌸           |
|   -----------------------------------------
|          石板路 / 小路                     |
+-------------------------------------------+
```

### 推荐配色

```
天空: #87CEEB (天蓝) → #B0E0E6 (渐变)
草地: #90EE90 (浅绿) → #228B22 (深绿)
石板路: #D3D3D3 (浅灰)
树木: #228B22 (深绿) + #8B4513 (棕色树干)
喷泉: #4682B4 (钢蓝) + #FFFFFF (水花)
```

## 4. 推荐实现路径
