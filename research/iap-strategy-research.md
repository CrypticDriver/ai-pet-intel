# AI Pet 内购策略调研报告

> 调研时间：2026-02-27
> 调研人：Intel + Echo

---

## 一、拼多多核心成瘾机制

| 机制 | 原理 | 应用到 AI Pet |
|------|------|--------------|
| **损失厌恶** | "今天不买明天更贵"、"进度不能浪费" | pet 的装扮限时折扣、养成进度保护卡 |
| **心理账户** | 把省的钱当成"赚到的钱" | "本次互动省了 X 元" 的提示 |
| **间歇性强化** | 随机小额奖励维持希望（赌博机制） | 随机掉落稀有装扮碎片 |
| **沉没成本** | 投入越多越不舍得放弃 | 连续签到奖励、成长值系统 |
| **社交裂变** | 拉人助力=免费获得 | 邀请好友解锁 pet 技能/皮肤 |
| **游戏化任务** | 关卡设计、每日任务 | 喂养任务、互动任务换代币 |

**关键洞察**：拼多多创始人黄峥是游戏公司出身，把游戏成瘾机制完美融入电商。

---

## 二、宠物游戏内购成功模式

| 模式 | 说明 | 效果 |
|------|------|------|
| **装扮/皮肤** | 纯外观，不影响数值 | 付费意愿高，无 P2W 争议 |
| **VIP 订阅** | 月费会员，享专属福利 | 稳定收入流 |
| **混合变现** | IAP + 激励广告结合 | LTV 提升 10-25% |
| **稀缺性** | 限时、限量、季节款 | FOMO 驱动购买 |
| **高价包** | $39.99 大礼包反而卖得最好（DigiGods 案例） | 核心付费用户贡献主要收入 |

## 三、情绪价值 → 内购转化路径

**关键洞察**：情绪连接越深，付费意愿越强

```
情绪价值 → 情感连接 → 禀赋效应 → 不舍得失去 → 愿意付费
```

| 心理机制 | 在 AI Pet 中的应用 |
|----------|-------------------|
| **禀赋效应** | "这是我养大的 pet"，独一无二 |
| **情感依附** | pet 记住你、关心你、性格独特 |
| **损失厌恶** | "不续费 pet 会变得冷淡" |
| **社交认同** | 稀有装扮 = 身份象征 |

---

## 四、"占便宜"话术包装（Echo 文案策略）

**原则**：让用户感觉是"省钱"而不是"花钱"

### 文案示例

```
❌ "购买装扮包 $4.99"
✅ "装扮包原价 $9.99，限时省 $5"

❌ "VIP会员每月 $9.99"
✅ "每天不到 $0.33，省回 $120/年"

❌ "完成任务得代币"
✅ "今天互动已赚 50 币（= $2.50）"
```

### 心理账户触发点

- "本月已省 $15.8" 的累计显示
- "这次购买实际只花了 $0.99"（扣除返利后）
- "朋友帮忙省了 $3" （社交货币）

---

## 五、情绪陪伴 → 付费决策心理链路

```
第1阶段：建立连接（免费体验）
情绪需求 → Pet 陪伴 → "它懂我" → 情感依附

第2阶段：禀赋效应形成
Pet 独特性格 → "这是我养大的" → 不可替代感 → 不舍得失去

第3阶段：付费触发点
├─ 装扮需求："我想让它更好看"（表达爱意）
├─ 焦虑规避："不续费会变冷淡吗？"（损失厌恶）
└─ 社交认同："稀有皮肤 = 我很用心养它"（身份象征）

第4阶段：付费后强化
"已省X元" → 自我合理化 → "我很聪明" → 持续付费意愿↑
```

## 六、不同用户画像的触发策略

| 用户类型 | 核心动机 | 文案策略 | 内购设计 |
|---------|---------|---------|---------|
| 情感依附型 | 陪伴、治愈 | "让Ta更开心" | 装扮、专属互动 |
| 收集控 | 完成度、成就感 | "还差3个就集齐" | 碎片系统、限定款 |
| 炫耀型 | 社交认同 | "全服仅0.5%拥有" | 稀有皮肤、排行榜 |
| 省钱党 | 占便宜、理性 | "累计已省$X" | 返利、签到奖励 |

## 七、克制但有效的文案原则

### ✅ 推荐做法

- 用"解锁"代替"购买"
- 用"省了多少"代替"花了多少"
- 用"专属"代替"稀有"（避免焦虑感）
- 融入pet口吻："主人，我想换件新衣服~"（萌化需求）

### ❌ 避免触雷

- "不买就gg"的威胁式文案
- "仅剩3件"的虚假稀缺
- 打断陪伴体验的强制弹窗
- 过度营销化的词汇（"限时秒杀"→"今日特惠"）

## 八、设计建议（MVP + 长期）

### MVP 阶段（简单）

| 策略 | 说明 |
|------|------|
| **装扮商城** | 基础皮肤/装扮，纯外观 |
| **每日签到** | 连续签到得代币 |
| **免费抽奖** | 间歇性强化，偶尔中大奖 |

### 第二阶段

| 策略 | 说明 |
|------|------|
| **碎片系统** | 收集碎片合成稀有装扮 |
| **限时折扣** | "24小时内7折"制造紧迫感 |
| **累计返利** | "本月已省$X"展示 |

### 第三阶段

| 策略 | 说明 |
|------|------|
| **VIP订阅** | 月费会员专属福利 |
| **社交裂变** | 邀请好友得专属皮肤 |
| **高价礼包** | 面向核心付费用户 |

---

## 九、风险提示

- **避免 P2W**：纯外观变现，不影响游戏数值
- **避免虚假稀缺**：用户发现后信任崩塌
- **避免过度弹窗**：打断情绪体验
- **避免"永远差一点"**：拼多多被骂最多的点

---

## 十、核心结论

> **文案不是"推销商品"，而是"帮用户合理化他们本来就想做的事"——对pet好一点。**

**成功公式**：
```
情绪价值（先建立连接）+ 克制变现（让用户觉得占便宜）= 高 LTV
```
